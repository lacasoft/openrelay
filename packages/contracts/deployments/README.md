# Deployments

Snapshots of deployed contract addresses per network.

## Files

- `sepolia.json` — Base Sepolia deployment (testnet, chain 84532)
- `mainnet.json` — Base mainnet deployment (TBD, blocked on external audit)

These files are generated by the deploy script + `make deploy-addresses`.
They are committed to the repo as a source of truth for merchants/SDK
integrators to look up current deployed addresses.

## Format

```json
{
  "chain": "base-sepolia",
  "chainId": 84532,
  "deployedAt": "2026-04-17T23:45:00Z",
  "deployer": "0x...",
  "contracts": {
    "stakeManager":    "0x...",
    "disputeResolver": "0x...",
    "nodeRegistry":    "0x..."
  },
  "usdc":     "0x036CbD53842c5426634e7929541eC2318f3dCF7e",
  "treasury": "0x..."
}
```
